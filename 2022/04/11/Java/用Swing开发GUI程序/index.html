用Swing开发GUI程序

发表于 2022-04-11 更新于 2022-04-21 分类于 Java
本文字数： 426 阅读时长 ≈ 1 分钟

Swing；Java中的一个报，负责开发GUI程序
GUI：图形用户界面，一般指可视化桌面系统中的应用程序
Windows视窗：从字符界面拓展到图形界面

常见图形界面基本结构：

  1. 界面（容器、窗口
  2. 控件（界面上的元素组件，如按钮、文本框等）
  3. 事件响应（相应功能）
  4. 渲染（颜色、字体）

界面

提供整个界面空间。

  1. 框架界面：javax.swing.JFrame
  2. 窗口界面：javax.swing.JWindow
  3. 对话框界面：javax.swing.JDialog

渲染

  • 背景颜色：setBackgrounp
  • 前景颜色：setForegrounp
  1. 颜色：java.awt.Color
  2. 字体：Java.awt.Font,但凡是有字的控件，都有setFont
