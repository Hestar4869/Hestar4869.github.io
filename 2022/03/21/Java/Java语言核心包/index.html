java.lang包

时Java中最基础的核心包，里面包含……

Java.lang.Math

Java.lang.String

直接赋值时，系统为了节省内存，用的是池机制

如何判断字符串内容是否相等？

  • 显然，不能使用 ==
  • 所以需要使用equals进行判断。

Java.lang.StringBuffer类

负责字符串处理，是自变型字符串，操作之后，自己变化。

Java.util包

最重要的是：Java集合框架（变长数组）
三类：

  1. List类：一维变长数组，元素有位置，可重复

    • ArrayList：底层用数组存储
    • LinkedList:底层用链表存储
    • Vector：实现了多线程的安全

  2. Set类：一维变长数组，元素无位置信息，不可重复

  3. Map类：二维变长数组

死锁
存储管理
